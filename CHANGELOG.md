# Changelog

---

## [1.0.4] – 2026-02-22

### عربي
- **إشعار انقطاع الإنترنت أثناء التحميل:** لو انقطع النت أثناء تحميل الصوت، يظهر للمستخدم تنبيه فوري بذلك.
- **حفظ التقدم تلقائيًا:** يتم حفظ آخر موضع وصل إليه التحميل والسور المختارة، حتى يتمكن المستخدم من الضغط على "أعِد المحاولة" ويكمل من حيث توقف.
- **تحديثات التطبيق عبر Firebase:** يتحقق التطبيق تلقائيًا من وجود إصدار جديد عند الفتح ويعرض نافذة تحديث أنيقة عند توفر تحديث.
- **تحسينات إشعارات الأذان:** تهيئة مبكرة لخدمة الأذان مع طلب صلاحيات الإشعارات والمنبّه الدقيق عند أول تشغيل.
- تحديث رقم الإصدار إلى **1.0.4**.

### English
- **Network disconnect alert during download:** If the internet is lost while downloading audio, the user is immediately shown a clear error message.
- **Auto-save progress:** The last downloaded position and selected surahs are saved, so the user can tap "Retry" and resume exactly where the download stopped.
- **Firebase-powered app updates:** The app automatically checks for new versions on launch and displays a premium update dialog when an update is available.
- **Adhan notification improvements:** Early initialization of the adhan service with upfront notification and exact-alarm permission requests on first run.
- Version bumped to **1.0.4**.
